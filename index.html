Open Source Contributions

Throughout my independant learning and research, I have heavily relied on open source software and tools. That being said, contributing and building such tools has become one of my favourite ways to build my software development skills, especially in the context of producing high quality code. Below are a few of my recent open source contributions, each linked with their associated PR or repo.

github.com/DSC-McMaster-U/Auto-ML
TL;DR: An automated machine learning service inspired by Google Cloud's Vertex AI and AutoML! This is a new project that I've designed as part of McMaster's Google Developer Student Club.
github.com/rawanmahdi/cnncam
TL;DR: Python library for generating visual explanations of convolutional neural network image classification, implementation of GradCAM proposed by R. Selvaraju. This codebase was also used to generate the experiments I implemented in my article that talks about GradCAM, so I wanted to publish it for reproducability of my results. I also wanted to implement my own GradCAM library to be used as a single line, making it easier for those just starting out with ML and XAI
github.com/NorskRegnesentral/shapr/pull/345
TL;DR: Shapley feature importance library that doesn't assume feature importance unlike kernel SHAP. I integrated native support for keras models. This was motivated by the fact that the library has the R backend that I needed for independant feature attributions but didnt support tensorflow keras models... so I made it support tensorflow keras models.

Therapy Bot (hackathon submission)
Language/Tools: Python, Tensorflow, TF Hub, pandas

Amazon Review Sentiment Analysis (personal project)
Language/Tools: Python, Tensorflow, TF Hub, TF Serving, Docker, RESTful API
Description: Fine-tuned a pretrained text embedding model via TensorFlow Hub to classify sentiments of customer product reviews. Simulated predictions via RESTful API, and cloud application deployment via TensorFlow Serving and Docker in manner efficient for scaling and versioning large models
Hand Written Digit Classifier using Numpy Alone (personal project)
Language/Tools: Python, NumPy
Description: Programmed MNIST hand written digit classifier without ML libraries (i.e. using mathematics of ML alone), implemeted gradient decent back propoagation and predictions with 85% accuracy. This project was a cornerstone to my foundational knowledge of the math behind heuristic machine learning.
Chest X-Ray Diagnostic Model (personal project, private repo)
Language/Tools: Python, Tensorflow, SciKit-Image
Description: Built program to process chest x-ray image data using pandas and scikit-image. Implemented ResNet machine learning model using Python (TensorFlow, scikit-learn) to train and predict diagnosis of diseased lungs with over 85 percent accuracy.
